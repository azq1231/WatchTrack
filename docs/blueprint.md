# **App Name**: WatchTrack

## Core Features:

- Phone Number Login: Allow users to log in using their phone number and OTP verification.
- Video Entry: Enable users to enter the name of a video and the current episode they are watching.
- Progress Tracking: Store and display the current episode for each video entered by the user.
- Video Search: Implement a search function that allows users to quickly find videos and their watch progress.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to provide a sense of focus and reliability, inspired by the tracking app's objective nature.
- Background color: Light Gray (#EEEEEE) - a muted shade of the primary hue, for a clean and unobtrusive backdrop that makes information pop.
- Accent color: Violet (#8E24AA) to guide users to calls to action or new features.
- Font pairing: 'Poppins' (sans-serif) for headlines and 'PT Sans' (sans-serif) for body text to give a modern and readable feel.
- Use clean, minimalist icons for navigation and video representation.
- Maintain a simple, intuitive layout with clear hierarchy for easy navigation and usage.
- Subtle animations when saving progress and displaying search results.
